# 출결 관리 시스템 - 서버

출결 관리 시스템의 서버입니다.

## 📂 프로젝트 구조

- `server.h`: 서버 헤더 파일
- `server.cpp`: 서버 구현 파일
- `*.json`: 사용자 정보 및 출결 정보 파일 (로컬에 저장됨)

## 🚀 실행 방법

```bash
# 서버 실행
git clone https://github.com/your-username/veda_project1_server.git
cd veda_project1_server
qmake
make
./veda_project1_server
```

## 🛠️ 기술 스택

- Qt 6.11.0
- C++
